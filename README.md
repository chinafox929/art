# 云上美术馆

一个带背景音乐的在线艺术博物馆网站，每日更新艺术推荐。

## 功能特点

- 🎨 **每日艺术推荐** - 每天精选一件艺术珍品，深度解读
- 🎵 **背景音乐** - 可选择古典音乐营造观展氛围
- 🏛️ **馆藏精品** - 按流派浏览印象派、文艺复兴、中国书画等
- 🖼️ **特展专区** - 主题展览，如"梵高与莫奈"
- 📅 **历史归档** - 查看往期推荐
- 🔍 **图片放大** - 点击作品查看高清大图

## 文件结构

```
art-museum/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # 交互脚本
└── README.md       # 说明文档
```

## 部署方式

1. 推送到 GitHub Pages
2. 或使用任意静态网站托管服务

## 每日更新内容

- `index.html` 中的今日推荐作品
- 历史归档添加新条目
- 可更新音乐列表

## 技术栈

- HTML5 + CSS3 + JavaScript
- 响应式设计，支持移动端
- 无框架依赖，轻量快速
